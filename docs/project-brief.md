# Project Brief — CCTV 모션 추출 및 객체 분석 시스템

## 비전
로컬 환경에서 작동하며(local-first), CCTV 영상 파일로부터 효율적으로 움직임(모션)이 있는 구간을 검출한 뒤 해당 구간에서만 객체(사람/차량 등)를 탐지하는 2단계 캐스케이드(Cascade) 모니터링 시스템입니다. 개인정보 유출 방지를 위해 외부 서버 업로드 없이 모든 분석을 로컬에서 수행합니다.

## 목표
- **2단계 분석 파라미터**: 값싼 모션 필터(DVR-Scan)로 99% 무의미한 구간을 걷어내고, 비싼 객체탐지(YOLO)는 모션 구간에만 1~2 fps 수준으로 실행하여 자원을 절약합니다.
- **배경 차분 결합 오탐 방지**: CCTV 화면 내에 정지해 있는 사람이나 차량(예: 주차된 차, 배경 인물)은 제외하고, 실제로 움직임이 발생하는 객체만 탐지합니다.
- **로컬 분석 및 프라이버시**: 원본 영상 및 분석 데이터를 외부(YouTube, Cloud Vision 등)로 절대 전송하지 않고 로컬 환경 내(`data/`)에서만 저장 및 분석합니다.
- **로컬 UI 제공**: 분석 결과를 탐색할 수 있는 직관적인 Streamlit UI를 제공하여 비전문가도 모션 타임라인과 추출된 썸네일, 실제 재생 및 객체 목록을 확인할 수 있게 합니다.

## 비목표 (Non-Goals)
- 원본/식별 가능한 영상을 외부 API 혹은 YouTube 등 외부 서비스로 업로드하여 분석하는 행위
- 고사양 실시간 스트리밍 분석(라이브 스트림 URL 직접 분석)
- 복잡한 다중 카메라 제어 및 대규모 VMS(Video Management System) 구축
